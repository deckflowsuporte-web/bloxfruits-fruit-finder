# Blox Fruits - Fruit Finder System

### Comando para colar no Delta:
```lua
loadstring(game:HttpGet("https://githubusercontent.com"))()
```

## Como usar:
1. Edite as configurações iniciais no arquivo `scripts/main.lua` com o seu nick e webhook.
2. Altere `SEU_USER` e `SEU_REPO` nas URLs para apontar para o seu repositório do GitHub pós-upload.
